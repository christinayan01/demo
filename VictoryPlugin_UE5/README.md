### インストール方法 How to use
C:\Program Files\Epic Games\UE_5.0\Engine\Plugins\Experimental\　フォルダに　VictoryBPLibrary　フォルダを置いてください。

Unreal Engine 5エディタを起動し、メニュの[Edit]→[Plugins]を開く。  
Victoryで検索して、VictoryPlugin機能を有効にします。  
エディタを再起動します。
