ドラえもんに出てくるひみつ道具「バイバイン」を再現するブループリントです。

# BaibainNewBP

### コンポーネント
StaticMeshにスポーンしたいスタティックメッシュをセットしてください。

### イベントグラフ
**Set Timer by Event** ：[Time]に何秒間隔でスポーンするかをセットします。  
**Spawn Actor** ：[Class]にBaibainNewBPをセットします。  

### ワールドに配置
BaibainNewBPをワールドに配置してください。

Playして確認してください。
